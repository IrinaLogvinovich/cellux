Форматирование кода

При первом ознакомлении с азами вёрстки кажется, что форматирование не влияет на качество вашего кода. Но это далеко не так. Хорошо отформатированный код легко читать, в нем можно быстро ориентироваться как автору, так и вашим наставникам и другим разработчикам, которые будут работать с вами в команде.

Форматирование не занимает много времени, зато правильно оформленный код выглядит аккуратно и презентабельно.

HTML страница должна иметь определенную структуру:

          
<!DOCTYPE html>
<html lang="ru">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My Document</title>
    </head>

    <body>
        Ваш код пишите тут
    </body>
</html>

Чтобы раскрыть такую заготовку в VS Code наберите ! и нажмите Tab.

Код разметки должен быть написан внутри тела страницы, тега body. При написании придерживайтесь нескольких простых правил:

  • Не пишите несколько тегов в строку, каждому тегу - своя строка, исключением из этого правила служат только теги, для форматирования текста, такие как i, em и подобные.

    
<div class="div"><p class="text">Текст <i>текст текст </i> текст</p></div>
            
    
<div class="div">
  <p class="text">Текст <i>текст текст </i> текст</p>
</div>

  • Соблюдайте вложенность элементов, каждый дочерний элемент должен быть смещет относительно родительского на два проблеы вправо

    
<div class="div">
<figure>
<img src="img/1.jpg" alt="скриншот с примером">
<figcaption>Описание картинки</figcaption>
</figure>
<ul>
<li>
<a href="#">Ссылка</a>
</li>
</ul>
</div>
            
    
<div class="div">
  <figure>
      <img src="img/1.jpg" alt="скриншот с примером">
      <figcaption>Описание картинки</figcaption>
  </figure>
  <ul>
      <li>
          <a href="#">Ссылка</a>
      </li>
  </ul>
</div>

  • Не злоупотребляйте пустыми строками, можно отделить пустой строкой крупные секции друг от друга, но не нужно вставлять пустую строку после каждого вашего тега.

    
<div class="div">

  <figure>
    <img src="img/1.jpg" alt="скриншот с примером">

    <figcaption>Описание картинки</figcaption>



  </figure>


</div>
            
    
<div class="div">
  <figure>
      <img src="img/1.jpg" alt="скриншот с примером">
      <figcaption>Описание картинки</figcaption>
  </figure>
</div>

  • Пишите тектовое содержимое тега в той же строке, в которой вы открываете тег и закрываете. Можно переносить текст только когда он очень длинный и не влезает в строку.

    

      
  
    • 
      Понедельник: 8:00 - 10:00
  
      • 
  
    • 
      Вторник: 8:00 - 10:00
  
      • 
  
    • 
      Среда: 8:00 - 10:00
  
      • 

    
            
    

      
  
    • Понедельник: 8:00 - 10:00
      • 
  
    • Вторник: 8:00 - 10:00
      • 
  
    • Среда: 8:00 - 10:00
